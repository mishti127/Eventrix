Campus Event Management System
Overview
Eventrix is a web-based campus event management system that helps students explore, register, and stay updated with various college events like technical, cultural, and upcoming activities.
It provides an organized platform for both students and organizers to manage events efficiently.

Features
 View all campus events (Technical, Cultural, Upcoming)
 Dedicated event pages for detailed information
 Event registration and login system
 Event reviews and feedback section
 Contact page for queries
 Dashboard for event overview
 Clean and responsive UI

Tech Stack
Frontend: HTML, CSS, JavaScript
Styling: Custom CSS (style.css, events.css)
Scripting: JavaScript (script.js)
Data Handling: XML (events.xml, events.xsl)

Future Enhancements
Backend integration (Node.js / Firebase)
Mobile responsiveness improvements
Smart event recommendations
Advanced analytics dashboard
